Study loop

Don’t memorize 100 random lines. Repeat the same clean tree until the first 8–12 moves feel automatic.

1. Play the tree

Use hint only when stuck. Try to remember the move first.

2. Replay mistakes

Undo and repeat the exact branch where you forgot the move.

3. Play rapid

Test the opening in real games. Save losses from the opening.

4. Add one branch

Only add new variations after the current ones stop feeling confusing.
